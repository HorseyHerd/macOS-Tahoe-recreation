# godot-on-linux
The community for running anything Godot on Linux.

I was doing research, and found that it would be possible to create a fully fletched, and working OS inside of the Godot Engine, export it to Linux, and load it as a default auto launch app, therefore replacing the actual Linux desktop, with your very own OS. There's only one issue. I may be very good with technology, and with Godot, but I have no experience building a Linux Distro.
So I can confirm, yes; it is possible. HOWEVER! Doing a such thing is an incredibly large and daunting task. And while yes, it is a very good, somewhat beginner friendly method to building your own OS, the amount of effort needed to do a such thing, stretches far. I will be getting to work on this process, but it will take years for anyone to be successful in this.
If you would like to join and help us, please, this is all open source. You may download the project and forget about it. If you do build off of it however, you must make it open source for Linux, and credit Godot as the desktop. And again, please feel free to do whatever you want with this.
